# Pappu Mining Windows Node

Official Windows package for running **Pappu Node** on Windows devices.

This package allows users to run Pappu Mining Node on:

* Windows 7
* Windows 10
* Windows 11

After installation, the system will generate a QR code. Scan the QR code to activate your device with your Pappu Mining activation code and connect it to your account.


---

## ✅ Minimum System Requirements

Before installation, make sure your PC meets the minimum requirements below:

| Requirement      | Minimum                             |
| ---------------- | ----------------------------------- |
| Operating System | Windows 7 / Windows 10 / Windows 11 |
| RAM              | 2GB minimum                         |
| Storage          | 20GB free space                     |
| CPU              | 2 Core CPU, 2.0GHz minimum          |
| Internet         | Stable internet connection          |

## ⚙️ Installation Steps

Follow these steps carefully to install and activate your Windows device.

### Step 1: Download ZIP

Click the download button from the official Pappu Mining website and save the Windows ZIP file on your PC.

Example file:

```text
windows.zip
```

---

### Step 2: Extract File

Right-click the ZIP file and extract it into a folder.

Example:

```text
C:\Users\YourName\Downloads\pappu-node-windows
```

---

### Step 3: Run as Administrator

Open the extracted folder.

Find the file:

```text
start.bat
```

Right-click on `start.bat` and select:

```text
Run as administrator
```

---

### Step 4: Wait for Setup

The setup process will start automatically.

Please wait until all tasks are completed.

After the setup finishes, the system will show a QR code on your PC screen.

---

### Step 5: Scan QR Code and Activate

Scan the QR code using your phone.

Then activate your device using your Pappu Mining activation code.

After activation, login or register with your Pappu Mining account.

---

### Step 6: Keep the Node Running

Keep the running process open on your PC.

Do not close the window while the node is running.

If you restart your PC, open the extracted folder again and run:

```text
start.bat
```

as administrator to start the node again.

## ⚠️ Important Notice

Keep the running process open on your PC.

Closing the process may stop the node.

After restarting your PC, you must run `start.bat` again as administrator.

After setup completes, scan the QR code and activate your device with your activation code.

---

## 🔐 Activation Process

The activation process works like this:

1. Run `start.bat` as administrator.
2. Wait until the setup completes.
3. QR code will appear on your PC.
4. Scan the QR code.
5. Enter your activation code.
6. Login or register with your Pappu Mining account.
7. Your Windows device will connect to your Pappu Mining account.

---

## ❓ Troubleshooting

### QR code is not showing

Make sure you ran `start.bat` as administrator.

Also check your internet connection.

---

### Node stopped after restart

This is normal.

After restarting your PC, open the extracted folder and run:

```text
start.bat
```

again as administrator.

---

### Windows blocks the file

Right-click on `start.bat`, choose **Run as administrator**, and allow permission if Windows asks.

---

### Internet connection issue

Make sure your PC has a stable internet connection before running the node.
